Нужно разработать небольшую страницу с использованием адаптивной верстки.
Не нужно создавать никакую бэкенд-архитектуру или интегрировать страницу в код на Python:
просто создать HTML-страничку, которую можно открыть в браузере.
Задача в том, чтобы красиво, лаконично и информативно
рассказать о себе потенциальному нанимателю и сверстать этот рассказ так, чтобы он открывался и
был читаем на поддерживаемых устройствах. Обязательно используйте фотографии.

Результат работы - https://olexiygolovko.github.io/cv_only_html_css/
